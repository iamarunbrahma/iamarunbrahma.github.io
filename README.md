# Portfolio Website

[Live Site](https://arunbrahma.com/)

A simple static portfolio website built with plain HTML and self-contained CSS - no
framework and no build step. Each page (`index.html`, `shelf.html`, `404.html`) is
standalone and styled in the classic minimalist style of an ML researcher's homepage,
with CSS-only dark mode that follows the visitor's system theme.

It is hosted on GitHub Pages at the custom domain `arunbrahma.com`. The
`.github/workflows/deploy.yml` workflow publishes the static files to the `gh-pages`
branch on every push to `main`.
